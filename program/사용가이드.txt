영단어 학습 프로그램 사용 가이드
================================

1. 실행 전에
- 프로그램과 함께 배포된 `영단어` 폴더 안에 있는 엑셀 파일(`.xlsx`)을 그대로 두세요.
- 엑셀을 열어 편집 중이면 저장이 되지 않을 수 있으니 학습 전에는 닫아 두는 것이 좋습니다.

2. 첫 화면(설정 창) 사용 방법
- `범위 선택 방식`에서 `챕터` 또는 `번호`를 고르고, 오른쪽 입력칸에 예시처럼 범위를 적습니다.
  · 챕터 예시: `1-3`, `2,5,8`
  · 번호 예시: `1-50`, `10,15,20`
- `미리 보기` 영역에서 불러올 단어 수와 필터 결과를 확인합니다.
- `학습 시작` 버튼을 클릭하거나 `Enter` 키를 누르면 학습 창으로 이동합니다.
- 설정을 취소하고 닫으려면 `Esc` 키를 누르세요.

3. 학습 창 사용 절차
- 화면 상단에 선택한 범위와 진행 상황(현재 step, 문제 수, 자동 저장 여부)이 표시됩니다.
- 문제를 확인한 뒤 `스페이스바` 또는 `Enter` 키를 눌러 정답을 확인합니다.
- 맞혔으면 `맞았어요` 버튼을 클릭하거나 `Y` 키를 누르고, 틀렸으면 `틀렸어요` 버튼 또는 `N` 키를 눌러 기록합니다.
- 자동 저장 기준(기본 10문제)에 도달하면 엑셀 파일에 저장되었다는 메시지가 상태표시줄에 나타납니다.
- 학습 중간에 범위를 다시 선택하려면 `범위 다시 설정` 버튼을 눌러 새로운 설정을 적용합니다.
- `학습 종료` 버튼을 클릭하면 마지막 저장과 함께 정답률이 높은 단어 목록(설정 시 표시)을 보여줍니다.

4. 단축키 정리
- 설정 창: `Enter` 학습 시작, `Esc` 취소/닫기
- 학습 창: `Space` 또는 `Enter` 정답 보기, `Y` 정답 기록, `N` 오답 기록
- 난이도(처음 등장 시): 숫자 `1~4` 또는 숫자 키패드 `1~4`

5. 프로그램이 단어를 고르는 방식(쉽게 설명)
- 처음 만나는 단어는 사용자가 난이도(매우 익숙~잘 모름)를 정하도록 요청합니다.
- 이후에는 틀린 횟수와 마지막으로 본 시점, 초기 난이도를 합쳐 "다시 볼 필요가 큰" 단어를 먼저 보여줍니다.
- 최근에 본 단어보다 오랫동안 안 본 단어, 자주 틀리는 단어가 더 자주 출제됩니다.
- 정답/오답을 기록할 때마다 누적 통계가 엑셀에 저장되어 다음 학습에 반영됩니다.
